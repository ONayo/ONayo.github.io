Listas

Ao invés de salvar nomes (exemplo) por várias váriaveis como:
nome1 = Pedro
nome2 = Rodrigo

Faça:
nomes = ['Pedro', 'Rodrigo']

Vamos pensar que você quer consultar a posição de algum nome na lista, para fazer isso, escreva no console:

"nomes[número escolhido]", ao invés de aparecer a letra nessa posição, aparecerá o nome nessa posição.

Pode escrever no console: "nomes[0][1]" para mostrar o primeiro caracter da segunda palavra.

Mesma coisa acontecerá com lista.length, vai falar quantos NOMES tem na lista.

É também possível subescrever algum elemento: nomes[0] - 'Murilo' / fazendo o Pedro ser subistituido por Murilo.

O "lista.push('algum nome')" e "lista.unshift('outro nome')" adicionam novos nomes à lista, porém o push adiciona no final, e o unshift no início.

O "lista.pop()" irá retirar algum do final e o "lista.shift()" será do começo.

Para retirar algum no meio da lista, escreva: "lista.splice(número, número2)", o primeiro número decide a partir de qual item você quer tirar, o número2 é quantos você quer tirar.

Também sendo possível escrever (por exemplo): "lista.splice(???, 0, algum nome)" retirando nenhum nome e adicionando um novo.

Tem como ver uma área dentro da lista, sendo "lista.slice(1,3)" visualizando o segundo e terceiro item da lista.

Para ver se algum item está na lista, escreva no console: "lista.includes('algum nome')", o console vai retornar para você com true ou false para caso o item exista na lista ou não.

Tem também o lista.indexOf('algum nome'), para saber a posição de algo (se der -1 é por que não tem na lista).

O lista.sort() vai reescrever tudo porém em ordem alfabética ou ordem crescente caso uma lista de número, e o lista.reverse() vai reverter a ordem atual.

Caso você queira que apareça todos os itens da lista na sua página, escreva: document.writeln(lista.join('')) sendo o que está dentro da '' é o que vai aparecer entre os itens.

É possível também escrever as coisas: "document.writeln('< b >'+lista.join('')+'< / b >')" (para escrever tudo em negrito) ou "document.writeln('< div >'+lista.join('')+'< / div >')" para escrever tudo dentro de uma div (espaçamento entre "b" e "< >" para não virar uma tag dentro do html enquanto estou explicando)
