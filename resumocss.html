Material De Estudo CSS

Início

A primeira coisa necessária é criar um outro arquivo porém com ".css" ou invéz de ".html".
Para todas as suas alterações no arquivo "css" funcionar, coloque "link" na cabeça do html e não troqu o que esta escrito no "rel" porém no "href" você deve colocar o nome do arquivo css.
Para você editar a escrita, você deve digitar o que você quer editar, depois colocar as flechas, e por fim colocar o que você quer fazer, por exemplo: "p{font-family: ;}", depois dos ":" coloque a fonte que você quer.

Tags Principais

Tags Com "font"

A primeira delas é "font-family" que serve para trocar a fonte da sua escrita.
A segunda é "font-size" que serve para trocar o tamanho da letra porém é preciso colocar "px" depois do número.

Tags De Cor

A primeira tag de cor é "color" que serve para trocar a cor das palavras, por exemplo: "color: red", passando o mouse por cima do quadrado colorido, você poderá trocar a cor para uma que você queira.
A segunda tag de cor é "background-color" por ser usado tanto no "body" para definir a cor dele e também nas tags de escrita com "p", "h1", "h2", etc, ela serve para trocar o fundo do corpo ou da escrita.

Tags De Div

As primeiras são "height" e "width", a primeira delas serve para trocar a altura do background-color, a segunda serve para trocar a largura.
A terceira é "border" e "border-radius", o "border" serve para criar um borda porém o "border-radius" serve para deixar a borda redonda, tendo os tipos de borda como solid, dashed, dotted, double, inset e outset.
Nós também temos os "border-right" e o "border-left" para mudar as bodas apenas nos lados para fazer em cima ou em baixo coloque "top" para cima e "bottom" para baixo.

Colocando quatro ou menos números em px no "border-radius" você pode mudar a borda de cada ponta do quadrado.
Você também pode colocar porcentagem no width como um tamanho do quadrado que vai se moldar de acordo com a página.
(para usar tudo isso você deve usar "div" e mudar as partes dele e para os números deve usar "px")

Usando "margin" ou "padding" você pode mudar a distancia do "div" com ele mesmo ou com o body, sendo o "margin" o espaçamento externo e o "padding" o espaçamento interno.
Também existe um alinhamento de texto que é o "text-align" dai é só escolher se vai ser no centro, esquerda, direita, etc, porém essa é uma maneira simples de fazer isso.

Colocando "class" dentro de uma imagem, você pode fazer vários tipos de imagens classificando elas.
Colocando "id" você pode colocar a mesma classe em um personagem mas também usar o id diferente para fazer alterações pequenas.
(Para usar o id escreva: "#algumacoisa" e na classe escreva: ".algumacoisa")


Colocando ":hover" em algum item no css, você pode fazer com que quanto você coloque o mouse por cima do item ele muda de acordo com o que você programou (precisa der 2 itens o "normal:hover" e o "normal").
Colocando "transition" no normal, você pode escolher o tempo de transição entre a imagem "Normal" e a "hover".
Também existe o "opacity" que serve para mudar a opacidade da imagem.


Colocando: ".algo > div{}", você pode editar div dentro de outras div, porém colocando ".algo > div > div{}", você pode editar div dentro de div


Tags de Flex

Colocando "display: flex; dentro de uma classe você pode usar "flex-direction: ???;" você pode usar para mudar o lugar onde os itens dentro dele estão usando "row" e "column".

Ao colocar "justify-content" dentro de uma classe, você poderá colocar os itens no centro, na esquerda, na direita entre outros (escreva em inglês e isso serve para colocar no centro, direita e esquerda em forma horizontal caso usar "row" em "flex-direction").

Ao colocar "gap" dentro de uma classe, você vai poder mudar o espaçamento entre os elementos dentro de uma div.

Ao usar "align-text" dentro da classe você pode mudar a direção dos itens em forma vertical caso você usar "row" em "flex-direction".

Você pode usar essas coisas dentro dos itens que estão dentro do principal por exemplo: ".algo > div" porém isso muda a direção das coisas dentro da div.

Para resolver o problema de quando passar o cursor por cima de algo que você programou para aumentar de tamanho e ele empurar os outros itens você deve usar: "scale" no lugar do "width" e "height". (use dentro do hover).


Você pode entrar no site Alura Flex para aprender mais sobre essas coisas de direções.
