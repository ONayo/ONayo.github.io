Parkour Collision

Descrição

O Parkour Collision é um jogo obby do roblox focado em usar a colisão para criar suas fases.

O Parkour Collision foi criado por Nayo com o objetivo de treinar mas ele decidiu fazer com que fosse um jogo, após um tempo surgiu mais pessoas para criar o jogo.

Notas

Este site foi feito apenas por Nayo, foi usado o Visual Studio Code para produzir o site e o GitHub para postar o site.
