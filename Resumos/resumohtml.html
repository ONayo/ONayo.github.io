Material De Estudo HTML

Início

Vamos para as coisas mais necessárias.
A primeira coisa para fazer é usar o "!" para criar o corpo básico do site.

head

A head do site é a parte onde você vai colocar as definições do site, ela e o body são necessários para um site.
A parte principal dessa parte é: "title" que é onde você vai colocar o título do site.
"link" que serve para você colocar um ícone ou imagens para o site (adicional), no "link" temos o "rel" e o "href", o rel você vai colocar o tipo de imagem que você quer, o href é o link da imagem que você quer colocar.
já no "style" você vai colocar as cores do site e o tamanho das letras do site, exemplo: h1{color: bronw; font-size: 35px}, body{background-color: gray;}, esses são apenas exemplos do que você pode fazer no style.

body

O body é a outra parte necessária do site, ele é onde você vai escrever todo o texto do site.

Escrita

Os parágrafos são o texto que você deve colocar ou seja para você criar um texto você deve digitar "p" e você escreve o que quer lá dentro.
Outra coisa muito útil é o "br" que serve para você colocar o texto para a próxima linha, quebrando no meio o texto.

Recursos De Escrita

Os recursos para o texto são as coisas escritas em Negrito, Itálico, etc.
O Negrito é primeiro deles, para usar ele você deve digitar "b", o "strong" é para deixar algo destacado, o "i" serve para colocar algo em Itálico, o "mark" serve para marcar como um marca texto, o "small" deixa o texto menor, o "del" serve para marcar algo "completado" como se você fizesse um risco para marcar que está pronto,

Recursos De Texto

Os recursos de texto são os usados para inserir link, imagem, fazer parágrafos e o estilo do texto.
Para você fazer um link você deve digitar "a" por exemplo: YouTube, lembre que o "href" é o link da imagem ou site.
para usar a imagem você deve colocar "img" o "alt" que fica dentro da tag img é para quando passar o mouse por cima você pode saber do que é.

Listas

Na parte de listas nós temos dois tipos: ordenada e não ordenada.
A diferença de cada lista é que a ordenada segue uma ordem de sua escolha usando "type" para escolher se vai colocar invertido, letras, números, etc.
A lista não ordenada serve para você usar quando vai colocar uma lista do mercado por exemplo.
Para usar as listas você deve usar "ol" para a ordenada e "ul" para a não ordenada, para criar um item da lista você deve usar "li".
Exemplo de lista:

    Comidas para comprar
  • Pão de carne
  • Cabeça da cauda da cobra
  • Queijo da Nine Boys
  • Carne estragada

Tabelas

Para usar as tabelas você deve digitar "table", recomendo dar espaço e depois border "table border", a segunda coisa que você deve usar nas tables é "tr" que é a linha da tabela, a terceira coisa que você deve colocar é "th" ou "td" que são os itens da tabela, o th fica destacado como um título ou categoria, colocando "colspan" é para você escolher o tamanho da coluna, colocando "rowlspan" você pode mudar o tamanho da linha. Exemplo de lista:

☢️Lista de compras☢️
Comida Bebida Extra
Concreto Àguinha da crystal Máscara do It aquela coisa
Quadro Branco Suco do lixo Mão da fantasia do It aquela coisa

Formulários

Para usar os formulários digite "form" para começar a programar isso, dentro do form digite "input" como item do formulário, também temos os tipos de input, temos "text" que só aceita texto, "number" que é só números, "password" é senha, "email" não preciso explicar, "range" que pode ser usado como volume, usado para personalizar idade, tamanha dos personagens entre outras coisas, "color" serve para colocar uma cor, "date" é para colocar uma data, "datetime-local" serve para colocar uma date com uma hora e lugar, "file" serve para colocar um arquivo e temos o "button" para ser como um botão de start, restart entre outros. Uma coisa que eu esqueci de citar, se você botar "target"dentro da tag de link, você poderá escolher se quando você entrar no link, para se ele vai criar uma nova aba ou se você vai substituir a aba atual.
por exemplo: eu boto para ir para o vídeo do never gonna give you up, botando "target"e dentro dele "_blank", o chrome vai criar uma nova aba para acessar o vídeo
