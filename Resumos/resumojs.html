Material de Estudo do Java Script

Começo

Para poder usar o Java Script, você deverá digitar (script src="") dentro do head (escrever o nome do arquivo nos aspas).
É muito importante citar o "//", que é os comentários do Java Script.
E caso digitar "/*" e depois "*/", podemos escolher exatamente onde está o comentário.
Temos as váriaveis também, que pode ser usada assim: "a = 1", "nome = algo", "cidade = 5", o primeiro item é a váriavel, o segundo item é a definição dela. (segundo item precisa estar entre aspas caso for apenas números não pode estar com aspas.)
Caso formos no "inspecionar" na página do site novamente, formos em console e digitarmos o nome da váriavel, irá aparecer a definição dele em baixo.
Conseguimos fazer calculos matemáticos com as váriaveis de número (definição que é número) no inspecionar (vezes = * divisão = /)

Calculos Matemáticos

Podemos escrever váriaveis nesse formato: "a = 5" ou "h = 15".
Dessa forma, podemos escrever "document.writeln(a+m), dentro do site, iria mostrar a resposta desse calculo (o "+" pode ser substituido por "*" para calculos de vezes, "/" para divisão e "-" para subtração.
Conseguimos escrever dentro do arquivo do java script a váriavel: "subtracao = a-m" (Não podemos usar acento ou "ç" para váriaveis ou tags), isso quer dizer que quando formos escrever "document.writeln("A resposta da subtração é "+subtracao)" irá aparecer "A resposta da subtração é -10" dentro do site. (-10 = 5 - 15)
Outras duas coisas são as potências e os módulos, os módulos são o resto de uma divisão (recomendável testar para entender), é usado "%" para ele. (10%3 = 1) (10/3 = 3,333...)
A potência já é conhecido, para usar ela deverá usar "**".
(As definições podem ser valores "boolean" que é verdadeiro ou falso)

Operadores de Comparação

Formas de comparar duas váriaveis de números, você irá botar no java script como: "a == b" (exemplo), você estará perguntando se o "a" é igual a "b", caso for sim, irá responder "true", caso for não, responderá "false".
Precisa usar o "alert" caso for fazer pelo java script a pergunta, ex.: alert(==)
Dá para fazer qual quer conta pelo alert também, mas essas são apenas por ali.

  • igualdade: "=="
  • maior que: ">"
  • menor que: "<"
  • maior ou igual: ">="
  • menor ou igual: "<="
  • diferente: "!="

Operadores Lógicos

Como o de comparação, esse também deve ser usado no alert ou no console do inspecionar.
O "&&" é usado para falar: "1==1 && 2==1" a resposta seria falso, por que apenas um deles é verdadeiro.
O "||" é usado para falar que um dois são verdadeiros: "1==1 || 2==1" a resposta seria true, mas se ambos estivessem errados, a resposta seria false.
O "!" é usado para reverter a resposta, "true --> false" e vice-versa.

  • e (and): &&
  • ou (or): || (para usar é usar a tecla na esquerda do "z" com shift segurado)
  • não (not): !

"Tags" no Java Script

document.

Podemos usar document.writeln("") para escrever no site como se fosse a tag p no html.
O document.writeln se torna útil por que podemos escrever dentro do arquivo js as váriaveis, botanto algo escrito no document.writeln (dentro do aspas) e depois do aspas escrevermos +nome da váriavel irá aparecer a definição da váriavel. Ex.: Anotando a váriavel nome = algo e no document.writeln("") botarmos dentro do aspas alguma coisa e depois do aspas +nome, irá ser escrito no site o "algo" depois do que você escreveu no document.writeln.
